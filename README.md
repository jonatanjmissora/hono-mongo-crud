Prerequisites:

- [Vercel CLI](https://vercel.com/docs/cli) installed globally

To develop locally:

```
npm install
vc dev
```

```
open http://localhost:3000
```

To build locally:

```
npm install
vc build
```

To deploy:

```
npm install
vc deploy
```


Teoria
=======

instalamos bun create hono@latest hono-mongo-crud
instalamos bun add mongoose zod@3.25.76
creamos el archivo .env
